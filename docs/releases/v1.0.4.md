# MX8 v1.0.4

## Highlights

- Coordinator HA is now the documented default posture, with failover contract and operator gates aligned to that behavior.
- Unified Python loader API is locked and documented (`mx8.load`, `mx8.image`, `mx8.video`, `mx8.mix`) with distributed behavior built in.
- `mx8.stats()` contract and docs are tightened so output remains simple, useful, and consistent.

## Reliability and Operations

- Added durable coordinator state store and replay paths for failover continuity.
- Expanded coordinator failover/elasticity gates and hardened state-store write behavior.
- Strengthened distributed resume behavior and checkpoint compatibility handling.

## Docs and Product Surface

- Added day-to-day user guide and aligned API docs with current runtime behavior.
- Synced HA/deployment/runbook language with current production posture.
- Updated crate versions to `1.0.4` for a consistent release surface.

## PyPI

- `mx8==1.0.4` is published.
